SLATE RELEASE NOTES

v1.0.21 — 2026-02-10
------------------------------
Hotfix: End-game screen crash

- Fixed crash on victory/defeat for players who had localStorage stats from
  older Slate versions that pre-date the ratingDistribution property.
  getStats() now merges stored data with defaults so new fields are always present.


v1.0.19 — 2026-02-10
------------------------------
Word Validation & Difficulty Rating

- Added 6,014-word English dictionary for guess validation
  • Guessing a non-word (e.g., random letters) → amber flash, "Not a valid word — no penalty!"
  • Guessing a real word that's wrong → red flash + penalty (unchanged behavior)
  • All 785 puzzle words included in valid set

- Added puzzle difficulty rating system
  • Each daily puzzle scored on 5 factors: unique consonants, vowel pattern ambiguity,
    consonant overlap, cascade potential, minimum keyboard consonants
  • 5 tiers: ⭐ Easy, ⭐⭐ Moderate, ⭐⭐⭐ Tricky, ⭐⭐⭐⭐ Hard, ⭐⭐⭐⭐⭐ Brutal
  • Shown on end-game results screen and already-played screen
  • Included in share text

- Synced sw.js CACHE_VERSION to v1.0.19
